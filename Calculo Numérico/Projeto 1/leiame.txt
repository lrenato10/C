PASSOS:

1- Compilar o codigo fonte "Codigo_fonte.c";

2- Executar "Codigo_fonte.exe";

3- Abrirá um console com um menu:
	Digite 0 para completar a resolução do problema proposto - perceba que as condiçoes iniciais e finais serão perguntadas, assim como o nome;

	Digite 1 para aparecer no console o resultado do teste  1 e gerar o "Relatorio do teste 1.txt" contendo dados complementares;

	Digite 2 para aparecer no console o resultado do teste  2 e gerar o "Relatorio do teste 2.txt" contendo dados complementares; 

	Digite 3 para aparecer no console o resultado do teste  3 e gerar o "Relatorio do teste 3.txt" contendo dados complementares;

	Digite 4 para rodar o teste extra, uma EDO analitica de segunda ordem é resolvida e então a ordem é testada... Dados complementares serão pedidos;

	Digite 5 para rodar o teste do numero de passos ideias para manter o delta energia menor que 1.0e-10 em um caso sem amortecimento. Dados extras serão requeridos;

	Digite qualquer outra tecla para sair do programa.

OBS.: Ao escolher alguma opção, caso seja necessário algum dado extra, aparecerá no prompt do console;

OBS.2: Nomes não devem conter acentos ou espaço entre as palavras;

OBS.3: Todos arquivos gerados, aparecem na mesma pasta do executável;

OBS.4: Todos os graficos foram plotados utilizando o GNUPLOT, seguindo comando:

gnuplot

gnuplot> set autoscale
gnuplot> set title 'Titulo'
gnuplot> set grid
gnuplot> set xlabel 'Legenda eixo x'
gnuplot> set ylabel 'Legenda eixo y'
gnuplot> plot 'Grafico.txt' u 1:2 with lines lw 2 title "Grafico 1", 'Grafico.txt' u 1:3 with lines lw 2 title "Grafico 2"
